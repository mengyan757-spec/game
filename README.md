# 智慧门禁守卫闯关游戏 🛡️

## 项目简介

这是一个专为**九年级苏科版信息科技课程**设计的在线教育小游戏，主题为《人脸识别 —— 我为校园增智慧》。游戏通过闯关形式让学生深度掌握人脸识别的三大核心步骤：**数据采集 → 模型训练 → 识别测试**。

## 🎮 游戏特色

### 三大关卡设计

#### 关卡一：【人脸数据采集筛选】📷
- 从20张校园人脸照片中筛选合格样本
- 包含多种质量类型：清晰正面照、逆光、遮挡口罩、模糊、侧脸、光线过暗等
- 学习目标：理解高质量、多样本是AI精准识别的基础

#### 关卡二：【AI模型训练调试】🤖
- 自主设置训练样本数量和迭代次数
- 实时观察训练进度、准确率曲线和Loss变化
- 三种预设模式：偷懒模式 / 标准模式 / 充足模式
- 学习目标：理解数据量、训练次数对模型性能的影响（AI三大要素具象化）

#### 关卡三：【智慧门禁实战识别】🚪
- 模拟真实门禁刷脸场景
- 包含10轮识别测试：本校学生、老师、陌生人、戴口罩人员、相似人脸等
- 3D门禁动画效果：舵机旋转开门、扫描线动画、红绿灯警示
- 学习目标：检验人脸识别系统实际效果，感知技术局限性

### 教学价值

✅ **知识目的**：掌握课本人脸识别三大核心步骤，理解数据质量对AI识别精度的影响  
✅ **能力目的**：将静态实验转化为动态操作任务，培养计算思维与AI模型调试能力

✅ **教学目的**：解决线上实验枯燥痛点，实现做中学、玩中学、测中学

## 🎯 游戏玩法指南

### 开始流程
1. 打开游戏 → 阅读任务说明 → 点击"开始挑战"
2. 进入关卡选择界面 → 按顺序完成三个关卡
3. 关卡需按顺序解锁，完成后可重玩

### 关卡一攻略
1. 观察20张照片的质量标签（合格/不合格）
2. 勾选12-15张清晰正面照作为训练样本
3. 避免选择模糊、遮挡、逆光、光线过暗的照片
4. 提交后查看知识点提示

### 关卡二攻略
1. 调整样本数量滑块（推荐200张标准模式）
2. 设置训练迭代次数（推荐50次标准训练）
3. 观察预估时间和预期准确率范围
4. 点击"开始训练"，观看实时训练过程
5. 训练完成后查看结果评价

### 关卡三攻略
1. 观察进入门禁的人员信息和AI识别结果
2. 判断AI的识别是否正确（放行/拦截是否合理）
3. 点击"识别正确"或"识别错误"按钮
4. 完成10轮测试后查看最终成绩和称号

### 评级标准
| 正确率 | 称号 |
|--------|------|
| ≥90% | 🏆 金牌门禁设计师 |
| 70-90% | 🥈 银牌门禁设计师 |
| <70% | 🥉 见习门禁管理员 |
